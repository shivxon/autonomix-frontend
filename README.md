# InsightBoard Frontend

React client for the InsightBoard Dependency Engine APIs.

## Setup
1. Install dependencies

```bash
npm install
```

2. Configure environment

```bash
cp .env.example .env
```

Update `VITE_API_BASE` if your backend runs somewhere else.

3. Run the dev server

```bash
npm run dev
```

Frontend runs at `http://localhost:5173` by default.

## Build

```bash
npm run build
npm run preview
```
